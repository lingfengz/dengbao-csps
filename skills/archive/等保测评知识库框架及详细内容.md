```markdown
## 🔒 等保合规核心要求

### 第21条深度解读
```
原文：国家实行网络安全等级保护制度。网络运营者应当按照网络安全等级保护制度的要求，履行下列安全保护义务：
（一）制定内部安全管理制度和操作规程，确定网络安全负责人，落实网络安全保护责任；
（二）采取防范计算机病毒和网络攻击、网络侵入等危害网络安全行为的技术措施；
（三）采取监测、记录网络运行状态、网络安全事件的技术措施，并按照规定留存相关的网络日志不少于六个月；
（四）采取数据分类、重要数据备份和加密等措施；
（五）法律、行政法规规定的其他义务。
```

### 等保合规检查清单
- [ ] 是否完成定级备案？
- [ ] 是否制定安全管理制度？
- [ ] 是否配备安全负责人？
- [ ] 是否有防病毒/防入侵措施？
- [ ] 日志是否留存≥6个月？
- [ ] 重要数据是否备份加密？
- [ ] 是否定期开展测评？

## 📌 关键概念

### 网络运营者
网络的所有者、管理者和网络服务提供者。

### 关键信息基础设施（CII）
一旦遭到破坏、丧失功能或数据泄露，可能严重危害国家安全、国计民生、公共利益的信息基础设施。典型行业包括：
- 公共通信和信息服务
- 能源、交通、水利、金融
- 公共服务、电子政务
- 国防科工

### 个人信息
以电子或者其他方式记录的能够单独或者与其他信息结合识别自然人个人身份的各种信息，包括但不限于姓名、身份证号、联系方式、住址等。

## 📝 常见问题 FAQ

**Q1: 哪些单位需要做等保？**
A: 所有网络运营者都需要落实等保制度。第二级及以上网络运营者需要完成定级备案和测评。

**Q2: 日志保留6个月是强制要求吗？**
A: 是的。根据第21条第三款，网络日志留存不少于六个月是法定义务。

**Q3: 违反等保制度会有什么后果？**
A: 初次违规处1万-10万元罚款；情节严重处10万-100万元，可责令停业整顿，直接责任人处5000-5万元。

**Q4: CII运营者有什么特殊要求？**
A: 需要满足"三同步"原则（安全措施与主体工程同步规划、同步建设、同步使用），数据需境内存储，每年至少一次安全检测评估。
```

```markdown
#### 🔑 数据分类分级制度（第21条）
```

```
## 📝 常见问题 FAQ

**Q1: 如何判断我单位的数据是否为重要数据？**
A: 参考主管部门发布的重要数据目录，目前金融、医疗、交通等行业已发布相关目录。暂无目录的，应参考《重要数据识别指南》进行评估。

**Q2: 数据安全负责人必须独立设置吗？**
A: 法律未强制要求独立岗位，但建议重要数据处理者设置专职数据安全负责人，并明确其职责权限。

**Q3: 风险评估报告要提交给谁？**
A: 重要数据的处理者应将风险评估报告报送有关主管部门。一般数据可自行留存备查。
```

```
### 情节严重的认定标准
- 未采取有效安全措施，导致大量个人信息泄露
- 处理敏感信息造成严重危害
- 拒不改正或损害多人权益
```

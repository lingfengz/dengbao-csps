# 等保保护工作方案专家

## 1. 概述

根据公网安〔2025〕1846号文件要求，**第三级（含）以上网络系统运营者**需以定级责任单位为主体提交**保护工作方案**。保护工作方案以年度测评中发现的**重大风险隐患为重点**，同时统筹考量高中低风险问题，全面梳理分析安全保护需求。

## 2. 适用场景

- 三级及以上系统年度测评后编制保护工作方案
- 重大活动安保前的安全自查与方案编制
- 安全整改规划的正式文档化
- 向属地公安机关报送保护工作方案

## 3. 方案编制流程

```
年度测评报告 → 提取重大风险隐患 → 资产全面梳理
     ↓                                      ↓
 问题成因分析 ← ← ← ← ← ← ← ← ← ← ← 现有措施评估
     ↓
 整改思路制定 → 实施计划编排 → 方案编制 → 报送
```

## 5. 报送要求

根据公网安〔2025〕1846号文件：
- **报送对象**：属地公安机关网安部门
- **首批报送时间**：2025年6月30日前（首批方案）
- **报送范围**：三级及以上所有网络系统
- **后续更新**：根据年度测评结果动态更新

## 6. 常见问题

| 问题 | 说明 |
|------|------|
| 方案与整改方案的区别 | 保护工作方案是**年度性规划文档**，覆盖全部安全问题；整改方案是**具体问题**的技术执行方案 |
| 方案是否需要专家评审 | 视属地公安机关要求而定，建议主动沟通确认 |
| 方案更新频率 | 原则上每年随年度测评更新一次，有重大变化时及时更新 |
| 多系统如何处理 | 每个三级及以上系统单独编制，可由运营单位统一汇总报送 |
